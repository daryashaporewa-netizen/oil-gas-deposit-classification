# Классификация мест залежей нефти и газа

Практическое задание №3 в рамках обучения в магистратуре МИФИ (направление «Машинное обучение»).

## Описание задачи
Разработка алгоритма машинного обучения для классификации места залежей (суша или море) на основе геологических параметров. Проект включает полный цикл: EDA, предобработку данных, подбор моделей и предсказание для соревнования.

## Данные и соревнование
Данные взяты из учебного соревнования на Kaggle:
[Classification of oil and gas](https://www.kaggle.com/competitions/classification-of-oil-and-gas/overview)

## Технологический стек:
* **Python** (Pandas, NumPy)
* **Scikit-learn** (Logistic Regression, модели классификации)
* **Matplotlib / Seaborn** (визуализация анализа данных)
